# Data Exploration and Analysis - Term Project
A completed term project for my Master's Degree. 

# Project Description: 
A project that takes you through the exploratory data analysis (EDA) process. 
A dataset is selected to apply various learned methods, functions, and analysis to the dataset throughout the course and then a summary of findings with a written summary and presentation of the data. 
A key outcome of the project will be a list of questions/hypotheses about the data at the start of the analysis of the data.
